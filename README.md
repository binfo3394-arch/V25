## 🚀 GitHub Pages Deploy — Step by Step

### Step 1 — Repository හදන්න
1. [github.com](https://github.com) → Sign in
2. **"New repository"** click කරන්න (+ icon, top right)
3. Repository name: `st-mobile-pos`
4. **Public** select කරන්න ✅
5. **"Create repository"** click කරන්න

### Step 2 — Files Upload කරන්න
1. **"uploading an existing file"** link click කරන්න
2. `index.html` සහ `README.md` drag & drop කරන්න
3. **"Commit changes"** click කරන්න

### Step 3 — GitHub Pages Enable කරන්න
1. Repository → **Settings** tab
2. Left sidebar → **Pages**
3. Source: **"Deploy from a branch"**
4. Branch: **main** | Folder: **/ (root)**
5. **Save** click කරන්න

### Step 4 — Live! 🎉
2-3 minutes පස්සේ:
```
https://YOUR-USERNAME.github.io/st-mobile-pos/
```

## 💾 Data Storage

Data **browser** ඇතුළේ store වෙනවා (IndexedDB + SQLite WASM):
- Browser clear නොකළොත් data safe ✅
- Regular backup ගන්න: **Settings → Download .db**
- Restore: **Settings → Restore .db File**
