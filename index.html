Firebase Project Setup:

1. console.firebase.google.com → New project
2. Build → Realtime Database → Create database (test mode)
3. Build → Authentication → Anonymous signin enable
4. Project Settings → Your apps → Web app → Copy config
5. Database Rules: { "rules": { ".read": true, ".write": true } }

Generated Peer ID → Parent dashboard Camera panel → paste → Connect.
